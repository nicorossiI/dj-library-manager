Procedura completa (prima volta)

  1. Apri Rekordbox
  2. File → Preferences → Advanced
  3. Sezione rekordbox xml → Imported Library → seleziona il file sopra
  4. Clicca OK e riavvia Rekordbox
  5. Nel pannello sinistro: clicca rekordbox xml
  6. Tasto destro su Playlists → Import Playlist
  7. Seleziona tutte le tracce → tasto destro → Import To Collection
Non spostare i file dopo l'importazione. Se cambi PC o lettera USB, rigenera rekordbox.xml.

Come funziona DJ Library Manager

Modalità Automatica Consigliata

  • Scarica la musica come al solito
  • L'app la processa da sola in background
  • Apri Rekordbox e clicca Ricarica (🔄)
  • Le canzoni sono già nelle playlist giuste

Modalità Manuale

  1. Trascina i file nell'app
  2. Clicca "FAI TUTTO IN AUTOMATICO"
  3. Aspetta che finisce
  4. Apri Rekordbox e clicca Ricarica

Doppioni

Il file con ★ CONSIGLIATO ha qualità migliore. Elimina gli altri manualmente usando il bottone "Apri cartella in Esplora File".

Rekordbox — come importare

  1. Apri Rekordbox
  2. File → Preferenze → Avanzate
  3. Sezione rekordbox xml → seleziona rekordbox.xml
  4. Clicca 🔄 nel pannello rekordbox xml
  5. Tasto destro → Import Playlist
Non spostare i file dopo l'importazione. Se cambi PC o lettera USB, rigenera rekordbox.xml.
